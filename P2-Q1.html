Funo de Especificidade e Sensibilidade

Sendo a sensitividade relacionada a proporo de verdadeiros positivos (capacidade do sistema de reconhecer dados positivos (no caso: sobreviventes)) - (VP)/(VP + FN)

e a especificidade relacionada a proporo de verdadeiros negativos (capacidade do sistema de reconhecer dados falsos (no caso: mortos)) - (VN)/(VN + FP)

def Sensitivity_specificity(confusion_matrix, model_name):
    print('Sensitividade do modelo {0}: {1:.3f}' .format(model_name, (conf[1][1])/(conf[1][0] + conf[1][1])))
    print('Especificidade do modelo {0}: {1:.3f}' .format(model_name, conf[0][0]/(conf[0][0] + conf[0][1])))
    return ''

XGBoost

- Algoritmo baseado em rvore. Em problemas envolvendo bases no estruturadas como imagens, algoritmos envolvendo redes neurais artificiais so considerados melhores

- Mas, no caso desse problema de pequeno porte com dados em tabelas algoritmos baseados em rvore de deciso so considerados melhores

- Modelo feito sequencialmente enquanto diminui o erro do modelo anterior enquanto aumenta a influncia dos melhores modelos

- Utiliza o algoritmo do gradiente (que minimiza o erro em modelos sequenciais) com processamento paralelo,

e vai podando as arvres de deciso que no so muito eficientes na classificao de instncias, o que reduzir a complexidade do classificador, assim melhorando a acurcia preditiva evitando overfitting

Anlise dos resultados

- Dada esta matriz de correlao podemos inferir os outros dados, temos que Matriz[0][0] = Verdadeiros negativos, Matriz[0][1] = Falso positivo

Matriz[1][1] = Verdadeiro positivo, matriz[1][0] = falso negativo

- A acurcia dada por: (Vp+Vn)/(Vp+Vn+Fp+Fn), mede a proximidade da medida para um determinado valor

- A preciso mede a prximidade da medida entre os dados, e dada por: (Vp)/(Vp+Fn)

- O parmetro f1 dado por: 2*((Precision)*(sensitividade)/(VN + FN)),

- f1 bom quando se deseja achar uma balana entre preciso e sensitividade e h uma distribuio muito desigual entre as classes
